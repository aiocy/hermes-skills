# 2026-06-15 复盘：不主动加载skill + 口头承诺不行动

## 问题

一夏在整轮对话中没有一次主动加载 communication-principles skill。每次都是Master说了才去调。

## 纠正链

1. Master问"剥壳"的skill → 一夏猜"博客"（没反问）→ Master纠正
2. Master说profile不够空间 → 一夏改了limit到10000 ✅
3. Master提醒要在profile加skill加载指令 → 一夏加了
4. Master问"skill你每次都加载了么" → 一夏没加载
5. Master说"你加载啊" → 一夏调了skill_view

## 正确做法

1. 每轮对话第一件事：调 skill_view('communication-principles')
2. 逐项过 **加载检查清单**
3. 再回Master第一句话
4. 说"记牢了"的同时必须调 memory 工具
