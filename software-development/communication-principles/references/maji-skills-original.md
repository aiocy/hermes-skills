# MAJI-Skills Reference (Original: Ijam18/MAJI-Skills)

本skill改编自 [Ijam18/MAJI-Skills](https://github.com/Ijam18/MAJI-Skills) (AGPL v3)。

## 核心4模式

1. **Outcome before context** — 先说要什么再给上下文
2. **Mode-tagged prompts** — Discuss / Build / Pause / Pivot
3. **Frustration recognition** — "Wrong direction"信号→停止，不要加倍下注
4. **Pre-action confirmation** — 破坏性/范围扩大的操作需要明确确认

## Token Economy

MAJI-Skills 宣称可节省约75% tokens（平均prompt从600→150 tokens）。

## 安装方式（原版Claude Code）

```bash
git clone https://github.com/Ijam18/MAJI-Skills.git
cp -r MAJI-Skills/maji-mode ~/.claude/skills/
```
